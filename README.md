
# 🔐 PassGuard – Password Strength Analyzer

PassGuard is a lightweight web application that helps users evaluate the strength of their passwords in real time.
It analyzes password complexity based on length, character diversity, and security best practices to encourage safer authentication habits.

---

## ✨ Features

* Real-time password strength evaluation
* Strength meter (Weak / Medium / Strong)
* Character validation checks
* Visual feedback with color indicators
* User suggestions for improving password security
* Clean and responsive UI

---

## 🛠 Technologies Used

* **HTML** – structure of the application
* **CSS** – styling and UI design
* **JavaScript** – password validation logic and dynamic feedback

---

## ⚙️ How It Works

PassGuard evaluates a password using several security criteria:

* Minimum length requirement
* Uppercase letters
* Lowercase letters
* Numbers
* Special characters

Based on these factors, the application calculates a score and displays a strength level.

Example:

| Password  | Result |
| --------- | ------ |
| abc123    | Medium |
| Abc@12345 | Strong |
